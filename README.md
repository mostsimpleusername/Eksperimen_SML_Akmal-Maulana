# Eksperimen SML - Telco Customer Churn

## Deskripsi
Project ini berisi proses eksperimen machine learning untuk prediksi customer churn menggunakan dataset Telco Customer Churn.

## Dataset
Dataset: Telco Customer Churn

## Tahapan Eksperimen
1. Data Loading
2. Data Cleaning
3. Data Preprocessing
4. Feature Engineering
5. Train-Test Split
6. Model Training
7. Model Evaluation

## Struktur Folder

```text
Eksperimen_SML
├── dataset-raw
├── preprocessing
├── requirements.txt
└── README.md
```

## Cara Menjalankan

Install dependency:

```bash
pip install -r requirements.txt
```

Jalankan preprocessing dan training sesuai script yang tersedia.

## Hasil

Model berhasil dilatih dan dievaluasi untuk memprediksi customer churn.
